# 批量邮件发送工具 - 启动说明

## 快速启动方法

### 方法1：使用批处理文件（推荐）

在项目目录下，直接运行：
```bash
启动.bat
```

> **提示**：如果是第一次运行（或在新电脑上），脚本会自动为您下载并安装所有必要的依赖库。这个过程可能需要几分钟，请耐心等待。

或者在任何位置，使用完整路径：
```bash
C:\Users\USER\sendemail\启动.bat
```

### 方法2：使用 npm 命令

在项目目录下运行：
```bash
npm run dev
```

### 方法3：创建桌面快捷方式

1. 右键点击 `启动.bat` 文件
2. 选择"发送到" -> "桌面快捷方式"
3. 双击桌面快捷方式即可启动

### 方法4：添加到系统 PATH（高级用户）

如果想在任何位置输入 `启动` 就能启动：

1. 右键点击"此电脑" -> "属性"
2. 点击"高级系统设置"
3. 点击"环境变量"
4. 在"用户变量"中找到 `Path`，点击"编辑"
5. 点击"新建"，添加：`C:\Users\USER\sendemail`
6. 点击"确定"保存
7. 重新打开终端，输入 `启动.bat` 即可

## 访问地址

启动成功后：
- 前端界面：http://localhost:3000
- 后端API：http://localhost:5001
- 局域网访问：http://192.168.101.233:3000

## 停止应用

### 方法1：一键停止（推荐）
双击运行项目目录下的 **`停止.bat`** 即可完全关闭所有相关服务。这能有效防止"端口被占用"的问题。

### 方法2：快捷键停止
在运行的终端窗口中按 `Ctrl + C`，然后输入 `Y` 确认即可停止应用。

### 注意事项
直接关闭黑色窗口有时可能导致后台服务未完全释放。如果遇到"无法启动"或"端口被占用"的问题，请运行 `停止.bat` 清理环境。

## 数据迁移说明

⚠️ **重要提示**：
**如果需要保留数据（客户名单、发送记录等），请手动复制项目根目录下的 `data` 文件夹到新电脑上。**
