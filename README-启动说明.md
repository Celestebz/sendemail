# 批量邮件发送工具 - 启动说明

## 快速启动方法

### 方法1：使用批处理文件（推荐）

在项目目录下，直接运行：
```bash
sendemail.bat
```

或者在任何位置，使用完整路径：
```bash
C:\Users\USER\sendemail\sendemail.bat
```

### 方法2：使用 npm 命令

在项目目录下运行：
```bash
npm run dev
```

### 方法3：创建桌面快捷方式

1. 右键点击 `sendemail.bat` 文件
2. 选择"发送到" -> "桌面快捷方式"
3. 双击桌面快捷方式即可启动

### 方法4：添加到系统 PATH（高级用户）

如果想在任何位置输入 `sendemail` 就能启动：

1. 右键点击"此电脑" -> "属性"
2. 点击"高级系统设置"
3. 点击"环境变量"
4. 在"用户变量"中找到 `Path`，点击"编辑"
5. 点击"新建"，添加：`C:\Users\USER\sendemail`
6. 点击"确定"保存
7. 重新打开终端，输入 `sendemail.bat` 即可

## 访问地址

启动成功后：
- 前端界面：http://localhost:3000
- 后端API：http://localhost:5001
- 局域网访问：http://192.168.101.233:3000

## 停止应用

在运行的终端窗口中按 `Ctrl + C` 即可停止应用。
