# 批量发推广信工具

一个面向企业用户的批量邮件营销工具，支持客户分组管理和个性化邮件发送。

## 功能特性

- 📧 企业邮箱SMTP/POP配置
- 👥 客户信息管理和分组
- 📋 邮件模板管理和变量替换
- 📤 批量邮件发送
- 📊 发送记录和统计分析
- 📁 支持附件功能
- 📥 导入导出Excel/CSV文件

## 技术栈

- **前端**: React + Ant Design
- **后端**: Node.js + Express
- **数据库**: SQLite
- **邮件服务**: Nodemailer

## 安装和运行

### 1. 安装依赖
```bash
npm run install-all
```

### 2. 启动开发服务器
```bash
npm run dev
```

前端将在 http://localhost:3000 运行
后端将在 http://localhost:5000 运行

### 3. 构建生产版本
```bash
npm run build
```

## 使用说明

1. **配置邮箱账户**：在设置页面配置企业邮箱的SMTP和POP信息
2. **管理客户**：添加客户信息，创建分组，或导入Excel/CSV文件
3. **创建模板**：编写邮件模板，支持变量替换功能
4. **批量发送**：选择客户和模板，预览后发送邮件
5. **查看记录**：在发送记录页面查看发送历史和统计信息

⚠️ **重要提示**：
**如果需要保留数据（客户名单、发送记录等），请手动复制项目根目录下的 `data` 文件夹到新电脑上。**
